About

MovieFinder is a lightweight, client-side demo that pulls data from the TMDB API and renders it with plain JavaScript—no frameworks. The app has simple tabs (Trending, Genres, About), an always-visible search bar, and a reusable renderList() + card builder pattern to keep the DOM updates clean and fast. Trending fetches weekly popular titles; Genres loads a responsive grid of genre buttons (which can route to Discover queries); and clicking a movie opens a slide-in details panel with backdrop, stats, overview, and top cast. It’s responsive, keyboard-friendly, and designed to be easy to extend (e.g., sort options, pagination, or watchlists) without changing the overall structure.
